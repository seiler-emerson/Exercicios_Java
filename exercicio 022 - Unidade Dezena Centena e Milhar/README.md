# Exercicio - Unidade Dezena Centena e Milhar
- Faça um programa que leia um número de 0 a 9999 e mostre na tela cada um dos dígitos separados.
- Exemplo: 3234

`
    Unidade: 4;
    Dezena: 3;
    Centena: 2;
    Milhar: 3;
`
