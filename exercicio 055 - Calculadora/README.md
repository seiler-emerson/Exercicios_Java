# Exercise - Calculator
- Create a program that reads two values and displays a menu on the screen:
   - [ 1 ] add
   - [ 2 ] multiply
   - [ 3 ] larger
   - [4] exit the program
- Your program must perform the requested operation in each case.

<details >

  - [ 2 ] multiplicar
  - [ 3 ] maior
